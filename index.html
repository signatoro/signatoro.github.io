Post Mortem

There were a lot of things that we wished we could have added to this engine if we had more time. If we had more time, we would have added more components, such as animation, physics, and lighting. Adding a particle effect system would also be a top priority, as it is a great component for any engine to have. Additionally, we would have liked to add an easy way to display text in the game, which would be great for making games like Undertale or Papers Please.

Some other things we would implement are data structures. Right now, we have a few data structures, but for the most part, we just use the given data structures. We would look into and start to implement more custom data structures. This would improve the flow and efficiency of our engine. After that, we would make a few more example games like Mario, Adventure, or Google Dinosaur Run to show the flexibility of our engine.
